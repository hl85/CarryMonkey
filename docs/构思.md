我希望搭建一个类似篡改猴（Tampermonkey）的浏览器插件——名叫搬运猴（CarryMonkey），用于帮助用户管理和运行自定义的JavaScript脚本，辅助用户批量处理很多繁重的工作。
该插件支持chrome mv3以上版本
### 核心功能：
1. **脚本管理**：
    - 用户可以添加、删除、启用/禁用自定义JavaScript脚本。
    - 脚本需要有明确的元数据（如名称、版本、描述等），可以通过注释块来定义。
    - 提供脚本编辑器，方便用户编写和修改脚本。

2. **脚本运行**：
    - 支持在指定网页上自动运行脚本。
    - 提供手动触发脚本执行的功能。
    - 脚本运行时能够访问页面的DOM和JavaScript环境。

3. **配置与设置**：
    - 允许用户对插件进行个性化配置，例如默认运行脚本、快捷键设置等。
    - 提供插件状态监控，显示当前已安装的脚本及其运行状态。

4. **用户体验**：
    - 界面简洁直观，易于操作。
    - 提供详细的日志记录，方便用户排查问题。

### 暂不实现功能
1. ***脚本仓库***
    - 暂时不实现脚本共享和下载功能，用户只能在本地管理脚本。

2. **安全性**：
    - 对用户上传的脚本进行安全检查，防止恶意代码执行。
    - 提供沙箱环境运行脚本，避免对用户浏览器造成影响。
